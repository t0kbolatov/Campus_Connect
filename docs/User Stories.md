# User Stories

## Story 1
As a student, I want to register an account so I can access the platform.  
Acceptance Criteria:
- unique email required
- password meets security rules

## Story 2
As a user, I want to log in so I can view my dashboard.  
Acceptance Criteria:
- valid credentials grant access
- invalid credentials show error

## Story 3
As a student, I want to view announcements so I stay informed.  
Acceptance Criteria:
- announcements load correctly

## Story 4
As a staff member, I want to create announcements so students receive updates.  
Acceptance Criteria:
- only staff can create announcements

## Story 5
As an admin, I want to manage user roles so permissions are enforced.  
Acceptance Criteria:
- role changes apply immediately
